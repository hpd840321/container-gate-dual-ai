# 版本变更日志

## v4.0.0 (2026-03-05)

### 重大改进

#### 1. 双20尺柜空间布局修正（核心修正）
- **问题**: 之前版本错误地将双20尺柜描述为"横向并排"
- **修正**: 改为正确的"纵向排列"（一前一后）
- **布局**: 车尾 → 后箱 → 前箱 → 车头
- **尺寸**: 总长12m（6m+6m），车道宽3.5m

#### 2. 车辆行驶方向修正
- **行驶方向**: 从左到右
- **车头位置**: 右侧（出口侧）
- **车尾位置**: 左侧（入口侧）
- **相机职责调整**:
  - C5（道闸前/左侧）: 拍摄车尾+后箱
  - C6（道闸后/右侧）: 拍摄前箱+车头
  - C3: 拍摄后箱侧面
  - C4: 拍摄前箱侧面

#### 3. SVG图形完全重绘
- **deployment_mode1.svg** (16KB): 侧壁安装模式，纵向双箱布局
- **deployment_mode2.svg** (18KB): 头尾安装模式，纵向双箱布局
- **camera_coverage.svg** (24KB): 视野覆盖与坐标映射
- **scenario_comparison.svg** (19KB): 三场景对比图
- 所有图形与文字描述完全一致

#### 4. 术语通用化
- 去除YOLOv8/YOLOv5特定技术术语
- 改用通用"AI目标检测"描述
- 提升方案的通用性和灵活性

### 更新文件
- index.html - 页面描述更新
- 道闸双柜识别方案.md - 技术文档全面更新
- 所有SVG流程图更新
