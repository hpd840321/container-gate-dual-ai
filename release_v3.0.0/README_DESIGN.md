## 🎨 UI/UX设计理念

作为资深 UI/UX 设计师，我在设计这些图表时深度考虑了以下美感要素：

### 色彩心理学应用
- **紫色渐变系** (#667eea → #764ba2)：传达科技感、专业性和创新性
- **粉红渐变系** (#f093fb → #f5576c)：用于标识相机设备，温暖且醒目
- **蓝青渐变系** (#4facfe → #00f2fe)：用于集装箱，冷静且专业
- **绿青渐变系** (#43e97b → #38f9d7)：用于单箱模式，清新且易于区分
- **暖色渐变系** (#fa709a → #fee140)：用于双箱模式，活泼且突出

### 视觉层次设计
1. **阴影效果**：每个元素都添加了柔和的投影（drop-shadow），增强立体感
2. **渐变填充**：避免单调的纯色，使用线性渐变增加视觉丰富度
3. **圆角处理**：所有矩形都采用圆角设计（rx 属性），更加亲和现代
4. **透明度叠加**：视野范围使用半透明填充，清晰展示覆盖关系

### 信息架构优化
- **模块化布局**：每个图表独立成模块，便于理解和复用
- **一致的视觉语言**：所有图表使用相同的色彩体系和设计风格
- **渐进式信息呈现**：从整体部署到细节原理，层次分明
- **图文结合**：每个图形元素都配有清晰的文字说明

### 用户体验细节
- **响应式设计**：SVG 矢量格式确保任意缩放不失真
- **交互导航**：HTML 页面提供标签页切换，快速定位感兴趣的内容
- **动画效果**：CSS 动画让页面加载更流畅自然
- **无障碍设计**：高对比度配色，确保可读性

## 🔧 自定义与扩展

### 修改颜色方案
SVG 文件中的颜色定义在 `<defs>` 部分的 `<linearGradient>` 元素中。您可以轻松修改这些渐变色以匹配您的品牌色彩。

例如：
```xml
<linearGradient id="topGrad" x1="0%" y1="0%" x2="0%" y2="100%">
  <stop offset="0%" style="stop-color:#667eea;stop-opacity:1" />
  <stop offset="100%" style="stop-color:#764ba2;stop-opacity:1" />
</linearGradient>
```

### 调整尺寸
SVG 是矢量格式，可以无损缩放。修改根元素的 `width` 和 `height` 属性即可：
```xml
<svg width="1600" height="900" xmlns="http://www.w3.org/2000/svg">
```

### 添加新元素
使用任何 SVG 编辑器（如 Adobe Illustrator、Inkscape、Figma）或直接编辑 XML 代码来添加新元素。
